# Time Report Application (with Notion API)

Web application for time reporting to a notion database, including different roles for
managers and users

## Getting Started

* npm
npm install npm@latest -g

### Built with

* node.js
* Express.js
* React
* React router
* React-Bootstrap

### Executing program


```
cd client
npm start
```
```
cd server
npm start
```
